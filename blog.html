Question-1: What are the different ways to select an element in the DOM?

DOM থেকে element নিতে পারি getElementById, querySelector, বা getElementsByClassName দিয়ে। এগুলো ব্যবহার করে HTML এর যেকোনো অংশে JavaScript দিয়ে কাজ করতে পারি।

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML HTML কোড সহ সব দেখায়।
innerText শুধু দেখা যায় এমন লেখা দেয়।
textContent সব লেখা দেখায়, এমনকি যেটা hidden সেটাও।

Question-3: What is event delegation in the DOM?

Event delegation মানে parent element কে event ধরার দায়িত্ব দেওয়া। এতে child গুলোর জন্য আলাদা করে event listener দিতে হয় না।

Question-4: What is event bubbling in the DOM?

Event bubbling হলো event নিচের element থেকে শুরু হয়ে উপরের দিকে যায়। যেমন, কোনো বাটনে ক্লিক করলে তার parent এর event ও trigger হতে পারে।

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript দিয়ে createElement() দিয়ে তৈরি করা যায়।
appendChild() দিয়ে HTML এ যোগ করা হয়, আর remove() দিয়ে মুছে ফেলা হয়।
